LidGuard Helper

Optional companion tool for advanced features

Download Helper (.pkg)

What It Adds

  • ● Motion detection — triggers theft mode when the Mac is picked up or carried (Apple Silicon only)
  • ● Lock screen overlay — displays your contact info on the device
  • ● Lid-close sleep prevention — keeps your Mac awake even with the lid closed
  • ● Power button detection — sends a Telegram alert right before an attempted shutdown

Installation

1
Download and open the .pkg installer above
2
Follow the installer — it places the helper in your Application Support folder
3
Open LidGuard — it connects to the helper automatically

LidGuard works fully without the helper. All core features — siren alarm, Telegram alerts, Bluetooth auto-arm — are built into the main app. Install the helper only if you need the extra features above.
